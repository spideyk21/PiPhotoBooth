PiPhotoBooth
==========
Raspberry Pi based Photo Booth

###Hardware:
-Raspberry Pi (any model should work)
-SD Card
-Raspberry Pi Camera or webcam
-Enclosure (optional)
-Wifi Module (optional)
-Monitor/screen
-Big Button

###Software:
-Raspbian (latest)
